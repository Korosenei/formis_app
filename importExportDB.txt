PowerShell 

Pour exporter la base de données:

& "C:\Program Files\PostgreSQL\15\bin\pg_dump.exe" -U postgres -h localhost -p 5432 -d formis -F p -f "C:\Users\$env:USERNAME\Documents\formis.sql"


Importer le fichier formis.sql

Étape 1 : Créer une base de données vide : 
& "C:\Program Files\PostgreSQL\15\bin\createdb.exe" -U postgres -h localhost -p 5432 formis

Étape 2 : Importer le fichier formis.sql dans cette base
& "C:\Program Files\PostgreSQL\15\bin\psql.exe" -U postgres -h localhost -p 5432 -d formis -f "C:\Users\$env:USERNAME\Documents\formis.sql"
